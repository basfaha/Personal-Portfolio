My Projects

Here are some of the projects I've worked on. Each one represents a unique challenge and learning experience.

Bookstore Archive

A project is a Java-based digital bookstore management system that handles inventory for books, CDs, and DVDs while tracking regular and premium memberships. It provides functionality for listing available products, managing a shopping cart, and processing member purchases.

System Pulse

A real-time system monitoring tool built with Python that displays live CPU, memory, and disk metrics alongside top processes in an elegant terminal dashboard. Uses psutil for metrics and Rich's Live display for flicker-free updates.

Snake Game

A classic snake game playable directly in the terminal. Built with Python using non-blocking keyboard input for smooth gameplay. Features collision detection, score tracking, and dual control schemes (WASD/arrow keys). A fun demonstration of game logic, data structures, and real-time user input handling.

Weather Forecast Tool

A CLI weather analyzer that fetches live forecast data via OpenWeatherMap API and performs statistical analysis on temperature trends. Features ASCII visualization, multi-city support, and secure API key management using environment variables. Built with Python and Rich for an elegant terminal interface

Donut in Blender

A photorealistic 3D donut created in Blender following the iconic Blender Guru beginner series. Demonstrates core 3D modeling skills including mesh manipulation, material creation with shader nodes, lighting setup, and final rendering. A foundational project showcasing proficiency with industry-standard 3D software.
